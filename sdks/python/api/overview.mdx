## Overview

`respan-api` is the REST API client for the Respan platform. It provides direct access to Respan's API for logging, evaluations, prompt management, and other platform features.

When used with the [`Respan`](/sdks/python/respan/overview) entry point, it is initialized automatically alongside `respan-tracing`.

```bash
pip install respan-api
```

Respan's Enterprise Platform provides better product performance and dedicated support for our enterprise customers. This guide will walk you through the setup process and help you integrate with our enterprise endpoints.

## Why enterprise platform?

- **Performance**: Faster loading speed and reliable service through our robust infrastructure
- **Dedicated support**: Direct access to founders for quick problem solving
- **Stable features**: Tested and reliable features that won't break your production systems

## 1. Sign up for enterprise platform
<Note>
Whether you're a new user or already have an account on our general platform, you need to create a new account on the enterprise platform.
</Note>
Access the Enterprise Platform through either: 
- [https://enterprise.respan.ai](https://enterprise.respan.ai)
- Respan login page: Select the `Enterprise portal` button.
<Frame>
<img src="https://keywordsai-static.s3.us-east-1.amazonaws.com/docs/ops/enterprise-portal.png" alt="Login page" />
</Frame>

### Code for logging in
Use the following code to log in to the enterprise platform:
`respan`.

## 2. API endpoint
We provide a separate API endpoint for enterprise customers. 
Use this endpoint: `endpoint.respan.ai` to send API requests.
### Example
**Logging API**: 

❌`https://api.respan.ai/api/request-logs/create/`
✅`https://endpoint.respan.ai/api/request-logs/create/`

Learn more about [Logging API](https://respan.ai/docs/documentation/getting-started/quickstart/logging).

**LLM proxy**

❌`https://api.respan.ai/api/chat/completions`
✅`https://endpoint.respan.ai/chat/completions`

Learn more about [LLM proxy](https://respan.ai/docs/documentation/getting-started/llm-inference).

## Will my plan transfer?
Your existing plan will automatically sync and transfer to the enterprise platform.

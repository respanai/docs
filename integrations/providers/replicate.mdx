1. **Sign up** — Create an account at [platform.respan.ai](https://platform.respan.ai)

</Accordion>

## Add Replicate API keys
There are 2 ways to add your Replicate credentials to your requests:

### Via UI
<Steps>
<Step title="Go to the Providers page">
Go to [Providers page](https://platform.respan.ai/platform/api/providers)
<Frame>
<img src="https://keywordsai-static.s3.us-east-1.amazonaws.com/docs/settings/providers.jpg" alt="Providers Page"/>
</Frame>
</Step>
<Step title="Add your Replicate credentials.">
Add your Replicate API key in the providers page.
</Step>
</Steps>

### Via code
Add `customer_credentials` parameter in your [request body](/apis/develop/gateway/chat-completions) To use your own Replicate credits.

```json
{
  // Rest of the request body
  "customer_credentials": {
    "replicate": {
      "api_key": "YOUR_REPLICATE_API_KEY",
    }
  }
}
```
